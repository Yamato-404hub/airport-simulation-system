# Airport Simulation System

## Overview
A multithreaded airport simulation system developed using Java concurrency concepts.

This project simulates airport operations such as:
- Aircraft landing
- Runway allocation
- Gate assignment
- Air traffic coordination

Developed as an academic group project at Asia Pacific University of Technology & Innovation (APU).

## Features
- Aircraft thread simulation
- Runway synchronization
- Gate allocation system
- Concurrent airport operations
- Shared resource management


## Tech Stack
- Java
- Multithreading
- Concurrency
- Object-Oriented Programming (OOP)


## System Components

### Airplane
Represents aircraft operations and flight behavior.

### AirTrafficController
Handles runway access and aircraft coordination.

### Runway
Controls synchronized landing and takeoff operations.

### Gate
Manages aircraft docking and gate assignment.

### RefuelTruck
Simulates airport ground service operations.
